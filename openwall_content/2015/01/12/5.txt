Title: Glance v2 API unrestricted path traversal through filesystem://        scheme Reporter: Jin Liu (EMC) Products: Glance Versions: up to and versions up to  Description: Jin Liu from EMC reported that path traversal vulnerabilities in Glance were not fully patched in OSSA By setting a malicious image location to a filesystem:// scheme an authenticated user can still download or delete any file on the Glance server for which the Glance process user has access to. Only setups using the Glance V2 API are affected by this flaw.
