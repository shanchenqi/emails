Status : Fixed  Description ================ Even though credentials are required to upload file into the server. any new combined vulnerability can allow an attacker to Upload shell into the server which gives entire root access of the server.  Technical Details ================     1. Logon into any wordpress application (localhost or public host)    2. Move on to Next Gen Gallery plugin file upload option available on    products.    3. Upload JPG file to the server through file upload option.    4. Modify the file variable contains JPG extension to PHP. Also, edit    and add shell content to the name variable body containing JPG    information/content.  Advisory Timeline ================ – Discovered in NextGen Gallery version. – Reported to plugins@...dpress.org – Vendor responded in the same. – Fixed in NextGen Gallery version.
