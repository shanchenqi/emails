Information ================================= Name: CSRF Vulnerability in TestLink Affected Software: TestLink Affected Versions: and possibly below Vendor Homepage: http://testlink.org/ Severity: High Status: Fixed   Vulnerability Type: ================================= Cross Site Request Forgery (CSRF)   CVE Reference: ================================= Not assigned   Technical Details: ================================= Even though the use of CSRF tokens are being implemented in the application, they aren't properly validated at the server side. This allows malicious requests to be generated by the attacker and get them processed by the server on behalf of the victim. By exploiting the vulnerability, the attacker will be able to create user accounts with administrator privileges on the application.
