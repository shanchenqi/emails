OVERVIEW  The vTiger CRM and lower versions are vulnerable to Remote Code Execution. No fixed version has been released as of   2. BACKGROUND  vtiger CRM is a free, full-featured, Open Source CRM software ideal for small and medium businesses, with low-cost product support available to production users that need reliable support. vtiger CRM is a widely used product with thousands of users in dozens of countries.  It has a vibrant community of users driving the product forward, and contributing to it's development.  Over 2 million copies of vtiger CRM have been downloaded so far. It was launched as a fork of version 1.0 of the SugarCRM project launched on December 31st,   3. VULNERABILITY DESCRIPTION  vTiger uses the vulnerable version of phpmailer class file located at /cron/class.phpmailer.php .   4. VERSIONS AFFECTED  Tested on   5. PROOF-OF-CONCEPT/EXPLOIT  File: /cron/class.phpmailer.php [code]     function SendmailSend($header, $body) {    if ($this->Sender != "")       $sendmail = sprintf("%s -oi -f %s -t", $this->Sendmail, $this->Sender);    else       $sendmail = sprintf("%s -oi -t", $this->Sendmail);  [/code]   6. SOLUTION  The vendor hasn't attempted to incorporate the latest version of phpMailer class in their vTigerCRM as of version  The flawed code portion can be patched with:  $sendmail = sprintf("%s -oi -f %s -t", escapeshellcmd($this->Sendmail), escapeshellarg($this->Sender)); $sendmail = sprintf("%s -oi -t", escapeshellcmd($this->Sendmail));   7. VENDOR  vTiger Development Team http://www.vtiger.com/   8. CREDIT  This vulnerability was discovered by Aung Khant, http://yehg.net, YGN Ethical Hacker Group, Myanmar.   9.
