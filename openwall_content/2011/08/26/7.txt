OVERVIEW  Jcow CMS versions  (4.x: 4.2 and lower, 5.x: 5.2 and lower) are vulnerable to Arbitrary Code Execution.   2. BACKGROUND  Jcow is a flexible Social Networking software written in PHP. It can help you to build a social network for your interests and passions, a member community for your existing website and a social networking site like facebook/myspace/twitter.   3. VULNERABILITY DESCRIPTION  The parameter "attachment" is not properly sanitized upon submission to /index.php, which allows attacker to execute arbitrary PHP code of his own.   4. VERSIONS AFFECTED  Free version:  4.x: 4.2 and lower Commercial version:  5.x: 5.2 and lower   5. PROOF-OF-CONCEPT/EXPLOIT   jcow file: /includes/libs/ss.inc.php line:         $app = $_POST['attachment'];    if (strlen($app) && $app != 'status') {        include_once('modules/'.$app.'/'.$app.'.php');        $c_run = $app.'::ajax_post();';        eval($c_run);        exit;    }   jcow file: /includes/libs/ss.inc.php line: 45  = $_POST['attachment']; if && != 'status') { = exit; }    6. SOLUTION  Free version users can upgrade to or higher. Commercial users can upgrade to 5.3 or higher.   7. VENDOR  Jcow CMS Development Team http://www.jcow.net   8.
