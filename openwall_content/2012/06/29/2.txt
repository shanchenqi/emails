The JLS Plugin  (jpeg_ls.dll) library fails to properly sanitize user-supplied input  resulting in a heap-based buffer overflow. With a specially crafted JLS  compressed image file, a context-dependent attacker could potentially  execute arbitrary code.  Impact ======  If a user could be enticed to open a malicious JLS file, the attack could  result in remote code execution.
