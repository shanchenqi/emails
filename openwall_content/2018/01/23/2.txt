It supports powerful and scalable directed graphs of data routing, transformation, and system mediation logic. For more information, see https://nifi.apache.org/security.html <https://nifi.apache.org/security.html>.  Apache NiFi host header poisoning issue  Severity: Medium  Versions Affected:  Apache NiFi -  Description: A malicious host header in an incoming HTTP request could cause NiFi to load resources from an external server.  Mitigation: The fix to sanitize host headers and compare to a controlled whitelist was applied on the Apache NiFi release. Users running a prior 1.x release should upgrade to the appropriate release.  Credit: This issue was discovered by Mike Cole.  Released: January 12,  Apache NiFi XSS issue in context path handling  Severity: Medium  Versions Affected:  Apache NiFi -  Description: A malicious X-ProxyContextPath or X-Forwarded-Context header containing external resources or embedded code could cause remote code execution.  Mitigation: The fix to properly handle these headers was applied on the Apache NiFi release. Users running a prior 1.x release should upgrade to the appropriate release.  Credit: This issue was discovered by Andy LoPresto.
