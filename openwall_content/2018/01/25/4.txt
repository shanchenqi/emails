Score: 5.3, AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:L Affected versions: 2.0 up to and Fixed versions: (not released yet), (not released yet)  We have identified a memory leak in Dovecot auth client used by login processes. The leak has impact in high performance configuration where same login processes are reused and can cause the process to crash due to memory exhaustion.  Patch to apply this issue can be found from  To our best knowledge, this patch should apply to all versions.  This issue can be mitigated on vulnerably systems by limiting login process to single request per process, which is also the default value.
