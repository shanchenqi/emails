Apache NiFi Java deserialization issue in template XML upload  Severity: Moderate  Versions Affected:  Apache NiFi -  Description: Any authenticated user (valid client certificate but without ACL permissions) could upload a template which contained malicious code and caused a denial of service via Java deserialization attack.  Mitigation: The fix to properly handle Java deserialization was applied on the Apache NiFi release. Users running a prior 1.x release should upgrade to the appropriate release.  Credit: This issue was discovered by Mike Cole.
