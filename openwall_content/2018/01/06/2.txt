(a) When the channel topic is set without specifying a sender, Irssi     may dereference NULL pointer. Found by Joseph Bisch.      was assigned to this issue.  (b) When using incomplete escape codes, Irssi may access data beyond     the end of the string. Found by Joseph Bisch.      was assigned to this issue.  (c) A calculation error in the completion code could cause a heap     buffer overflow when completing certain strings. Found     by Joseph Bisch.      was assigned to this issue.  (d) When using an incomplete variable argument, Irssi may access data     beyond the end of the string. Found by Joseph Bisch.      was assigned to this issue.   Impact ------  May affect the stability of Irssi.   Affected versions -----------------  (a,b,c,d) All Irssi versions that we observed.   Fixed in --------  Irssi   Recommended action ------------------  Upgrade to Irssi Irssi is a maintenance release in the 1.0 series, without any new features.  After installing the updated packages, one can issue the /upgrade command to load the new binary. TLS connections will require /reconnect.
