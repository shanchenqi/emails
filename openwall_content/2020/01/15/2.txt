However this configuration is not respected and the certificate verification disables trust verification in every case. This exclusion also gets registered globally which disables trust checking for any code running in the same JVM.  Mitigation: Users of the affected versions should apply one of the following mitigations: - Upgrade to Apache Beam or later  Acknowledgements: This issue was reported (and fixed) by Colm Ó hÉigeartaigh.
