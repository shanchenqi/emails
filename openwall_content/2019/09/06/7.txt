Given a username for a user that can be authenticated via LDAP, it is possible to improperly authenticate as that user without that user's correct password.  Mitigation: 3.x users should upgrade to If the upgrade cannot be done immediately, LDAP authentication can be disabled by removing the Traffic Ops LDAP configuration file -- ldap.conf -- in order to mitigate the vulnerability until an upgrade to can be performed.
