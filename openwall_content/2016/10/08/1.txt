Subject: ffmpeg before    ff_draw_pc_font in ffmpeg before uses incorrect font_height value ,which allows remote attackers to cause ffmpeg service a out-of-bounds array write fault when it uses 'ansi' to decode a AVI file which has a crafted 'strf' struct.      avi_read_nikon in ffmpeg before is vulnerable to infinite loop when it decode an AVI file which has a crafted 'nctg' struct.       i2f in ffmpeg before uses incorrect re_signal value ,which results in an out-of-bounds array read .       cavs_idct8_add_c in ffmpeg before uses incorrect block value ,which results in an out-of-bounds array read .       avi_read_header in ffmpeg before is vulnerable to a memory leakage issue when it decodes an AVI file which has a crafted 'strh' struct.     avi_read_seek in ffmpeg before uses incorrect scale value , which allows remote attackers to cause a assert fault of service via an AVI file which has a craft 'strh' struct.       read_gab2_sub in ffmpeg before is vulnerable to a null-point-exception when it decodes an AVI file which has a crafted 'gab2' struct.
