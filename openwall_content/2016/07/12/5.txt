I have recently found three vulnerabilities in ws-xmlrpc library - https://ws.apache.org/xmlrpc/. Apache Security Team have assigned three CVE numbers for Apache Archiva project as it uses ws-xmlrpc library.  Here is the list of vulnerabilities with CVE numbers:     - - SSRF attack via loading external DTD in ws-xmlrpc.    - - Deserialization of untrusted data via serializable    data type in ws-xmlrpc.    - - DoS attack via Content-Encoding header in ws-xmlrpc.
