The framework is designed to streamline the full development cycle, from building, to deploying, to maintaining applications over time.  This release contains fixes for the following potential security vulnerabilities: - A crafted JSON request can be used to perform a DoS attack when using the Struts REST plugin   - Vulnerability in the Jackson JSON library    Please read the Version Notes to find more details about performed bug fixes and improvements. The 2.5.x series of the Apache Struts framework has a minimum requirement of the following specification versions: Servlet API 2.4, JSP API 2.0, and Java 7.
